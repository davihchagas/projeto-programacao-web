Bem-vindo ao portal

Notícias, perfis e áreas de administração em um único site estático.

Este projeto foi montado apenas com HTML e CSS, seguindo o fluxo de telas apresentado. A proposta é mostrar navegação entre áreas públicas, autor, leitor, editor e superadmin, com exemplos de conteúdo e formulários.
